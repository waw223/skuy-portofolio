# Portofolio Skuy

Website portfolio statis untuk layanan edit foto dan video.

## Isi Folder
- `index.html` — halaman utama
- `styles.css` — gaya dan tata letak
- `script.js` — interaktivitas tema, slider, dan video
- `assets/` — gambar dan video pendukung

## Cara Menjalankan
1. Buka folder ini di editor kode seperti VS Code
2. Buka `index.html` di browser

Atau gunakan ekstensi Live Server di VS Code untuk melihat perubahan secara real-time.

## Cara Membagikan di GitHub
1. Pastikan Git sudah terpasang di komputer
2. Buka terminal di folder ini
3. Jalankan perintah:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/<username>/<repo>.git
   git push -u origin main
   ```
4. Jika ingin menggunakan GitHub Pages, aktifkan pada pengaturan repositori.

## GitHub Pages
- Setelah push ke GitHub, buka halaman repositori.
- Masuk ke `Settings` > `Pages`.
- Pilih branch `main` dan folder `/root`.
- Simpan dan tunggu beberapa menit hingga situs aktif.
- Alamat akan menjadi `https://<username>.github.io/<repo>/`.

## Catatan
- Ganti `https://github.com/<username>/<repo>.git` dengan URL repo GitHub kamu sendiri.
- Sesuaikan konten `index.html`, `styles.css`, dan `script.js` jika ingin mengubah layanan dan portofolio.
