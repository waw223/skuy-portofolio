Caranya order gimana?

Klik tombol WhatsApp, kirim brief + referensi + deadline. Nanti saya balas estimasi biaya & waktu.

Butuh file apa aja?

Untuk video: footage + musik referensi (opsional) + contoh style. Untuk foto: foto original resolusi terbaik.

Revisi berapa kali?

Tergantung paket. Umumnya 1–2x revisi minor. Revisi besar/ubah konsep bisa kena biaya tambahan.

Siap bikin kontenmu naik level?
